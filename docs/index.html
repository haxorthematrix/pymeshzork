examine zorkmesh

ZorkMesh is a multiplayer implementation of the classic Zork text adventure, connected via Meshtastic radio mesh networks or MQTT over the internet. Explore the dungeon together, chat with fellow adventurers, and watch out for grues!

  • Classic Zork gameplay with Z-Machine interpreter
  • Multiplayer via Meshtastic LoRa mesh radio
  • See other players in your current room
  • Chat with nearby adventurers
  • OLED display support for portable play
  • MQTT bridge for internet connectivity
  • Play offline via radio mesh, or online via MQTT - no Meshtastic hardware required!
  • Build your own adventures with the included map editor
